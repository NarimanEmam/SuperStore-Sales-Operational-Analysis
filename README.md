# SuperStore Sales & Operational Analysis

📌 **Project Overview**  
This project analyzes the SuperStore dataset to evaluate sales performance, profitability, and return behavior using a complete ETL workflow in Excel.  
The goal was to connect revenue growth with operational efficiency and identify areas affecting profit margins.

🔧 **Tools & Process**  
- Power Query (ETL & Data Cleaning)  
- Data Modeling  
- KPI Development  
- Pivot Tables  
- Interactive Excel Dashboard  

📊 **Key Focus Areas**  
- Sales Growth Trend  
- Profit & Margin Analysis  
- Category Performance  
- Returns Impact  
- Regional & State-Level Insights  
- Shipping Mode Analysis  

📈 **Business Questions Answered**  
- Which category drives the highest profit?  
- Where are returns affecting margins?  
- Which region performs best?  
- Does shipping mode influence return rate?  

📂 **Files Included**  
- Excel Dashboard File (`SuperStore_Dashboard.xlsx`)  
- Raw Data File (`SuperStore_RawData.xlsx`)  
- Dashboard Screenshots (`Screenshots/`)  

🎯 **Outcome**  
This project strengthened my ability to transform raw data into structured analysis and actionable business insights.
